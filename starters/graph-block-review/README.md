# Graph Block Review Starter

This starter contains static graph and block metadata for local review.

It demonstrates:

- a descriptor-only graph/block bundle;
- a typed block manifest;
- a diagnostic-only execution ledger;
- workspace artifact metadata;
- an API drift metadata gate;
- a generated fixture manifest.

These files are examples for local validation. They do not run graph nodes, start workers, install packages, fetch artifacts, open files, generate client code, approve resources, certify safety, or publish anything.
