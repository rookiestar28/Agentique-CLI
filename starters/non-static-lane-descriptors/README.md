# Non-Static Lane Descriptors Starter

This starter shows how to describe several Agentique resource lanes with static public metadata.

The files in `lanes/` are examples for:

- agent cards and descriptors
- external endpoint registrations
- downloadable packages
- tool-enabled package listings
- static skill and workflow packages
- hosted-deferred public readback records

These examples are preparation material only. They do not publish, approve, certify, host, execute, proxy, or moderate resources. Platform upload, scan, review, moderation, publication state, distribution state, and public readback are handled by `agentique.io`.

Keep these descriptors public and inspectable. Do not add tokens, cookies, non-public endpoint values, local paths, generated archives, dependency folders, executable payloads, or personal data.
