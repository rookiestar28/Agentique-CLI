# Parser Variant Import Review Starter

This starter shows a static parser and variant metadata example for local review. It uses synthetic workflow metadata and source-only variant availability.

The files are inspectable text and JSON. They do not run a workflow, install packages, call a service, start a server, build a container, open a notebook, or publish a resource.

Local validation checks package shape and public metadata boundaries. Upload review, platform scan, publication state, distribution state, and readback are handled by `agentique.io`.
