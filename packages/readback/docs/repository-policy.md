# Repository Policy

This repository contains read-only public readback clients and badge helpers.

Release is blocked until:

- Owner approval exists.
- Public-content review passes.
- Secret scanning passes.
- SDK and badge tests pass.
- Stale and unavailable states are verified.
- Method inventory review confirms no publish, edit, delete, admin, moderation, private scan, or report mutation helpers exist.

Intermediate local git history may be replaced before public launch. The final public history must start from reviewed public-safe content only.
